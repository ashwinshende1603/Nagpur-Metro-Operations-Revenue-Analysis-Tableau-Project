# 🚇 Nagpur Metro Operations & Revenue Analysis | Tableau Project
![nagpur](https://metrorailnews.in/wp-content/uploads/2018/05/Nagpur-Metro-Train.jpg)

## 📌 Project Summary

This project analyzes operational performance, passenger trends, revenue distribution, and cost efficiency of the Nagpur Metro Rail system using Tableau.

The objective is to deliver actionable business insights through interactive dashboards and data storytelling.

---

## 🛠 Tools & Technologies

- Tableau Desktop
- Microsoft Excel
- Data Modeling (Relationships using Trip_ID)
- Calculated Fields
- KPI Cards
- Parameters & Filters
- Dashboard Actions
- Data Visualization Best Practices

---

## 📊 Dataset Overview

The dataset contains 5,000+ records across four related tables:

- Trip_Details
- Station_Operations
- Train_Performance
- Revenue_Summary

Key fields used:
- Trip_Date
- Corridor
- Station_Name
- Ticket_Type
- Passenger_Count
- Net_Revenue_INR
- Operational_Cost_INR
- Maintenance_Cost_INR
- Delay_Min
- Energy_Consumption_kWh

## 📖 Storytelling Approach

The analysis is structured into three themes:
- Growth & Demand Evolution
- Revenue & Financial Performance
- Operational Efficiency

---

## 🎯 Key Business Insights

- Peak hours significantly influence ridership and operational load.
- Revenue performance varies by corridor.
- Operational costs differ substantially across stations.
- Maintenance expenses show periodic trends aligned with operational intensity.
